Hospital Management System (Java Swing + LinkedList)
Description

This Hospital Management System is a Java Swing-based desktop application that helps manage basic hospital patient records.
It uses an in-memory LinkedList to store and manage patient data — no external database required.

Users can easily add, view, update, and delete patient details through a modern, user-friendly graphical interface.

Features

Add Patient – Register a new patient by entering ID, name, age, disease, and doctor.
View Patients – Display all stored patient records in a scrollable text area.Update Patient – Modify existing patient details by entering the same patient ID.
Delete Patient – Remove a patient record using their unique ID.
Clear Fields – Instantly clear input fields to add or edit new data.
LinkedList Storage – Uses Java’s LinkedList data structure for fast, dynamic storage.
Modern Swing GUI – Professional-looking interface with clear layout and color styling.

Concepts Used

Object-Oriented Programming (OOP)

Java Swing (GUI Development)

Event Handling (ActionListeners)

Data Structures (LinkedList)

Encapsulation and CRUD Operations
